跟着这份教程，用 OmniFocus 3 搭建任务管理系统

13小时前

少数派付费组

目录
  1. ${ item.innerText }

想做好任务管理，有好工具是不够的，还要知道如何用它去搭建一套完整的任务管理系统。

OmniFocus 3 中加入了标签，增强了自定义透视，让它俨然成为一个没有短板的任务管理工具。不过要怎么使用它，才能搭建出一套高效的任务管理系统呢，这是本教程想探讨的问题。

对于还不熟悉 OmniFocus 的你，这是一份从入门开始的教程，不去费事费力地钻研枯燥的理论和使用说明，而是从应用的操作开始实践任务管理；对于已经在使用 OmniFocus 的你，这可以是一本工具书，深入细致地解析 OmniFocus 的每一个功能，力求不错过应用的每一种可能性。

以 GTD 理论作为背景知识和线索，我们将从应用最基本的功能出发，探索自定义透视等 OmniFocus 的核心功能，在此过程中不断尝试它与 AppleScript 和 URL Schemes 的结合，抵达 OmniFocus 使用效率的边界。

这份《用 OmniFocus 3 搭建任务管理系统》，希望帮助你梳理使用 OmniFocus 的方法论，搭建出属于你自己的任务管理系统。

你将学到

  1. 系统的任务管理理论：搞懂任务管理的每一个环节，让你从零入门任务管理。
  2. 实用的任务管理经验：每个步骤都有实践中总结出的建议，让你不再走弯路。
  3. 细致的 OmniFocus 功能介绍：OmniFocus 3 所有的功能介绍和分析，搞懂每一个细节。
  4. 紧跟新功能：标签、自定义透视等新功能的剖析，帮你快速梳理 OmniFocus 的新用法。
  5. 进阶 OmniFocus：不会代码也会耍魔法，用实例理解和学习 AppleScript 和 URL Schemes。
